# Portfólio Lucas Dias (jogável)

Este repositório contém um scaffold inicial para um portfólio em formato de jogo 2D (side-scroller) usando Next.js (App Router), TypeScript, Tailwind CSS e Phaser 3.

Instalação

1. Instale dependências

   npm install

2. Rodar em desenvolvimento

   npm run dev

3. Build

   npm run build

Deploy

- Conecte o repositório no Vercel e execute o deploy (Next.js é suportado nativamente).

Observações

- O scaffold inicial cria cenas básicas do Phaser com assets gerados dinamicamente. Ainda há muito a implementar (animações, inimigos avançados, itens colecionáveis, modais ricos, som, partículas). Use a branch feat/portfolio-game para continuar o desenvolvimento.
